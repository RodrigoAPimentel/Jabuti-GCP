- É necessario a instalação do banco de dados Mysql;
- Configuração inicial do banco de dados:
	# usuario: root,
	# senha: toor.
- O banco de dados por ter configurações personalizadas no arquivo application.properties em src/main/resources.
- Servidor TomCat configurado para a porta 8090;
- UML incluido no root do projeto com o nome "Jabuti-GPC(UML)";
- Arquivos de Log são armazenados na pasta "logs" no root do projeto.
